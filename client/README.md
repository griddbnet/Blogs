## Run Locally

Clone the project

```bash
  git clone https://github.com/Babajide777/grid-db-meal-planner.git
```

Go to the project directory

```bash
 cd client
```

Install dependencies

```bash
 npm install
```

Start the client

```bash
  npm start
```
